This was made in order to update certain arcade cabinets in the USA in order to bring them in line with new custom scripts. This is not meant to update regular arcade cabinets.
Portions of this script have been edited out in order to preserve the privacy of the individuals who wanted this script to be made. 
This means the script will not work as is and the portions marked EDITED in the script need to be changed in work. 
If you are using this, please ensure to changes the edited variables and filepaths in both extractor.py and in main.c.

Proper use of this script:
project.exe and extractor.py should be placed in a zip and be uploaded to the proper drive url. The existing script in the arcade cabinet will download and extraxt the zip adn then run the files contained within.
